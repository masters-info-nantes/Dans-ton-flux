#Dans-ton-flux

##Compile & run

###Install Interfaces

In the Dans-ton-flux directory :
```
$ cd InterfacesClientServer
$ mvn install
```

### Run server

In the Dans-ton-flux directory :
```
$ cd Server
$ mvn compile exec:java
```

### Run client

In the Dans-ton-flux directory :
```
$ cd Client
$ mvn compile exec:java
```
